 Info Tutorial - Cara Menghitung Value Zein 

 Apa itu Value Zein?

Value Zein adalah ukuran kekuatan dalam dunia **Devil GEROKSAR**, dihitung berdasarkan kombinasi **Devil Star** dan **Groin**dan**RPG POWER.

 Rumus Perhitungan Jika Seseorang Memiliki Void Star dan Groin

{Value Zein} = {Zein Void Star}) x {Tahap Groin})

 Rumus Perhitungan Jika Seseorang Hanya Memiliki Void Star

{Value Zein} = {Zein Void Star}) x (TINGKAT RPG POWER)

 Rumus Perhitungan Jika Seseorang Hanya Memiliki GROIN

{Value Zein} = {Tahap Groin}) x 500

 Contoh Perhitungan penggun Void star + Groin

Jika seseorang memiliki **Devil Star Super (10.000 Zein)** dan **Groin tahap 2**, maka:

Value Zein = 10.000 x 2 (tahap Groin)

Value Zein = 20.000 Value Zein

 Contoh Perhitungan Pengguna Groin Murni

Jika seseorang memiliki **Groin tahap 4**, maka:

Value Zein = 500 x 4 (tahap Groin)

Value Zein = 2.000 Value Zein

 Contoh Perhitungan

Jika seseorang memiliki **Devil Star Ultra (2.500 Zein)** dan Memiliki RPG Power Tingkat 20, maka:

Value Zein = 10.000 x 20

Value Zein = 200.000 value Zein
